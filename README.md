# SinoSDK

    SinoSDK 是三诺生物传感股份有限公司的蓝牙连接的SDK.
    
# 如何安装

##1.使用CocoaPods安装
推荐使用这种安装方式。
step1:添加下面一条语句到Podfile:
````
pod 'SinoSDK','~> 1.0.6'
````
step2:导入.h文件
````objc
#import "SinoSDK/SinoSDK.h"
````

##2.手动安装
不推荐。

# 如何使用

请查看SinoSDKDemo.
